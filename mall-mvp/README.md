## 促销模块
在电商和O2O领域，促销是运营人员的一个主要的让利行为，同时促销活动期间的购买量也较之普通商品更高，不同的阶段，对于促销的要求也是不同的。

促销实质上是一种沟通活动，即营销者（信息提供者或发送者）发出作为刺激消费的各种信息，把信息传递到一个或更多的目标对象（即信息接受者，如听众、观众、读者、消费者或用户等），以影响其态度和行为。
